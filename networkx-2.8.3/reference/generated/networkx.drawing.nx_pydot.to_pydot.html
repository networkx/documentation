to_pydot#

to_pydot(N)[source]#

Returns a pydot graph from a NetworkX graph N.

Parameters
NNetworkX graph

A graph created with NetworkX

Examples

>>> K5 = nx.complete_graph(5)
>>> P = nx.nx_pydot.to_pydot(K5)
