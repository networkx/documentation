networkx.utils.misc.nodes_equal¶

nodes_equal(nodes1, nodes2)[source]¶

Check if nodes are equal.

Equality here means equal as Python objects. Node data must match if included. The order of nodes is not relevant.

Parameters
nodes1, nodes2iterables of nodes, or (node, datadict) tuples
Returns
bool

True if nodes are equal, False otherwise.
