graphs_equal(graph1, graph2)[source]#

Check if graphs are equal.

Equality here means equal as Python objects (not isomorphism). Node, edge and graph data must match.

Parameters:
graph1, graph2graph
Returns:
bool

True if graphs are equal, False otherwise.
