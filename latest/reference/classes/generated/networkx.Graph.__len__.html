Graph.__len__#

Graph.__len__()[source]#

Returns the number of nodes in the graph. Use: ‘len(G)’.

Returns:
nnodesint

The number of nodes in the graph.

See also

number_of_nodes

identical method

order

identical method

Examples

>>> G = nx.path_graph(4)  # or DiGraph, MultiGraph, MultiDiGraph, etc
>>> len(G)
4
